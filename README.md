# 🔥 쇼츠 살포기

> **나만 쇼츠 보면 나만 망한다.<br>다같이 보면? 상대평가에서 안 떨어진다.**

시험 기간, 유튜브 쇼츠를 끊을 수 없는 당신을 위한 **상호확증파괴 시스템**.

쇼츠를 보다가 "이건 나만 보기 아깝다" 싶으면 **살포** 버튼을 누르세요.
같은 확장 프로그램을 설치한 모든 사람의 브라우저에서, 지금 보고 있는 웹 페이지의 이미지가 당신이 보내준 쇼츠로 바뀝니다.

교수님 강의자료? 쇼츠.
네이버 뉴스 썸네일? 쇼츠.
도서관에서 열심히 정리하던 노션? 쇼츠.

**모두가 참여자이자, 공격자이자, 피해자.**

---

## 뭘 살포하든 자유

- 🐱 고양이가 박스에 쏙 들어가는 영상
- 💪 새벽 4시에 일어나서 운동하는 동기부여 영상
- 🍜 진짜 맛있어 보이는 먹방
- 🎵 한번 들으면 하루종일 머릿속에 맴도는 노래
- 🚫 수능 금지곡

나만 보기 아까운 영상을 마음껏 뿌리고, 서로의 시간을 태워버리세요.

---

## 작동 원리

```
1. 쇼츠를 본다
2. "이거 다같이 봐야 해" 라는 생각이 든다
3. 🔥 살포하기 버튼을 누른다
4. 모두의 브라우저에서 이미지가 하나씩 쇼츠로 바뀐다
5. 누군가는 교수님 강의자료를 보다가 쇼츠를 보게 된다
6. 그 사람도 쇼츠를 살포한다
7. 무한 반복
```

---

## 시스템 구성

| 구성 요소 | 설명 |
|-----------|------|
| **Chrome 확장 프로그램** | 살포 버튼 + 피격 시 미디어 교체 |
| **실시간 서버** | WebSocket 기반 살포 메시지 중계 |
| **대시보드** | 실시간 살포 현황판 — 누가 얼마나 뿌렸는지, 누가 가장 많이 맞았는지 |
| **랜딩 페이지** | 설치 가이드 (3단계면 끝) |

---

## 설치 방법

**30초면 됩니다.**

1. [여기서 ZIP 다운로드](#) → 압축 해제
2. 크롬 주소창에 `chrome://extensions` 입력 → 우측 상단 **개발자 모드** ON
3. **"압축해제된 확장 프로그램을 로드합니다"** 클릭 → 폴더 선택

끝. 이제 유튜브 쇼츠로 가서 🔥 버튼을 찾으세요.

## 대시보드 미리보기

실시간으로 업데이트되는 **살포 현황판**:

- 🟢 현재 접속자 수
- 🔥 총 살포 횟수
- 💥 총 피격 횟수
- 🏆 살포왕 / 피격왕 리더보드
- 🌐 네트워크 그래프 — 누가 누구를 공격했는지 실시간 시각화
